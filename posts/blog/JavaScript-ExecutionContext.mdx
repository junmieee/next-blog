```
1. _git rm .env --cached_ 
```
해당 파일을 추적에서 제거한다. --cached 옵션은 워킹 디렉토리에 있는 파일은 삭제하지 않고, 인덱스(index)에서만 추적을 제거함으로로컬 파일은 그대로 유지되며 Git의 추적에서만 제거된다.
```
2. _git commit -m '커밋 메시지' 
```
변경 사항을 커밋해 추적에서 제거된 파일들이 Git의 히스토리에서 사라지게 된다.
