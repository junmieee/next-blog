if 조건문을 통해서 소괄호 사이에 조건문을 추가하게 되면 그 조건의 값이 참인 경우 코드가 실행됨

### 자바스크립트의 Truthy & Falsy는 boolean데이터에만 국한되지 않는다.

아래의 코드를 실행해보자. 마찬가지로 참!이 출력된다. 

```jsx
//참과 거짓(Truthy & Falsy)

if (123) {
    console.log('참!')
};
```

![스크린샷 2023-01-13 오후 7.52.02.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/d743cbbe-d1cb-45f8-92fc-02b9a194025a/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA_2023-01-13_%E1%84%8B%E1%85%A9%E1%84%92%E1%85%AE_7.52.02.png)

‘거짓(Falsy)’에 해당하는 값은 아래와 같다. 

1) false

2) 0 

3) null

4) undifined

5) NaN

6) ‘ ’(빈 문자)

7) 0n

```jsx
const fruits = ['Apple', 'Banana'];

if (fruits) {
    console.log('아이템이 들어있음!')
};
```
