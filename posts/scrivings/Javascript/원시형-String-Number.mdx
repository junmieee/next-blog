### 부동소수점 오류

아래와 같은 코드를 검사해보자. 

```jsx
const a = 0.1;
const b = 0.2;

console.log(a + b);
```

→ 0.3이라는 결과가 나올 것으로 예상되는데, 실제로 콘솔에는 아래와 같이 출력된다. 

![스크린샷 2023-01-13 오후 12.14.58.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/ca8478d0-ed4e-4690-87a7-3abc228cbd0e/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA_2023-01-13_%E1%84%8B%E1%85%A9%E1%84%92%E1%85%AE_12.14.58.png)

컴퓨터는 기본적으로 0,1의 이진수를 사용해서 동작하게 됨. 숫자를 표현하려면 10진수를 써야하는데 컴퓨터가 10진수를 2진수로 표현해야 하기 때문에 무한소수라는 개념이 발생하게 되고 이것을 유한하게 표현하기 위해서 세부적인 값에 초과나 손실로 계산 오류가 발생하게 됨

이를 해결하기 위해 아래와 같이 toFixed()라는 매소드를 호출함. 인수로는 원하는 소수점 자리를 입력

```jsx
console.log((a + b).toFixed(1));
```

그러나 위와 같이 출력하면 문자 데이터로 출력되게 됨. 

숫자데이터로 바꿀 때는 ‘Number()’함수를 호출 해 숫자 데이터로 바꿔준다. 

```jsx
console.log(Number((a + b).toFixed()));
```
