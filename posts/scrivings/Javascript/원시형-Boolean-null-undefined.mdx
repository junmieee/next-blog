## Boolean

Boolean은 true와 false값을 이용하는 논리데이터이다. true→ 긍정, false→ 부정

```jsx
 Boolean(불린)

const a = true; 

const b = false;
```

아래의 코드를 실행시 콘솔에 ‘Hello’라는 문자열이 출력된다. 

```jsx
if(a) {
    console.log('Hello')
};
```

하지만, 아래와 같은 코드를 실행 시 콘솔창에는 아무것도 출력되지 않는다. 

```jsx
if(b) {
    console.log('Hello')
};
```

## Null, Undefined

- null: 존재하지 않거나 값이 비어있음. 혹은 값을 알 수 없음

현재 age라는 값은 명시적으로 값이 없는 상태임

```jsx
let age = null;

console.log(null);
```

- undefined: null과 같이 값이 없는 상태지만 암시적으로 값이 없는 상태이다. → 즉 직접 undefined라는 값을 할당하지 않아도 undefined라는 값이 존재하고 있는 것임.

아래와 같은 코드를 실행해 보자. 

```jsx
const user = {
    name: "Jem",
    age: 85
};

console.log(user.name);  // Chloe
console.log(user.age);  // 85
console.log(user.email); //undefined
```

user 객체에 Email속성이 없는데도 불구하고속성을 조회하려고 시도하니 이것을 암시적으로 undefined로 출력함
