New(생성자 함수)와 Array라는 함수를 통해서 3개의 문자데이터를 가지고 있는 배열 데이터를 만들어 보자. 

```jsx
// Array(배열)

const fruits = new Array('Apple', 'Bannana', 'Cherry')

console.log(fruits);
```

아래와 같은 방법으로 배열을 만들 수도 있다. 

```jsx
// Array(배열)

const fruits = ['Apple', 'Banana','Cherry']

console.log(fruits);
```

### 배열 데이터 인덱싱

배열 데이터에서 대괄호를 통해 숫자로 배열데이터를 조회하는 것을 배열 데이터를 인덱싱한다고 말한다. 

```jsx
// Array(배열)

const fruits = ['Apple', 'Banana', 'Cherry']

console.log(fruits[1]);
```

'Apple', 'Banana', 'Cherry’ 는 각각 배열의 아이템 또느 요소(Element)라고 한다.
