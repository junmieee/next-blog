# Node.Js 란?
    
## Node.js는 Chrome V8 JavaScript 엔진으로 빌드된 **JavaScript 런타임**
    
    *런타임? 프로그래밍 언어가 동작하는 환경
    
    *JavaScript가 동작할 수 있는 환경이 두지 있음
    
    1) Node Js가 설치되어 있는 특정한 컴퓨터 환경
    
    2)크롬과 같은 기존의 웹브라우저
    
    웹브라우저에서는 HTML, SCC, JS만 동작한다. 이것만 가지고 개발하면 비효율적임. 따라서 개발을 도와주는 여러가지 모듈들을 가져와서 도움을 받아가면서 개발을 하게 됨. 하지만 이러한 모듈들은 브라우저에서 직접적으로 동작할 수 없기때문에 대표적으로 Node.Js환경에서 도움을 받은 내용들을 HTML SCC JS로 변환을 해 주어야 함. 변환작업에 대한 명령을 내려줘야 하고 이 변환작업이 돌아갈 수 있는 환경이 필요함. 따라서 우리는 JS라는 프로그래밍 언어로 이러한 변환을 만들어줄 수 있음.
